General notes

Task manager API to abstract the connection and logic with the mongo database

API detail

Task Collection

READ all tasks

Obtains a JSON with all the tasks on the system

  • Curl
  • HTTP
curl -X GET "{{url}}/tasks"
GET %7B%7Burl%7D%7D/tasks HTTP/1.1
Host:

INSERT a task

Inserts a task on the system based on a JSON document provided and returns the result of this operation in JSON format

  • Curl
  • HTTP
curl -X POST -H "Content-Type: application/json" -d '        {"title": "test title",
        "reference": "test ref",
        "description": "test desc",
        "timeWorked": [],
        "status": "test status",
        "visible": true }' "{{url}}/tasks"
POST %7B%7Burl%7D%7D/tasks HTTP/1.1
Host: 
Content-Type: application/json

        {"title": "test title",
        "reference": "test ref",
        "description": "test desc",
        "timeWorked": [],
        "status": "test status",
        "visible": true }

DELETE a task

  • Curl
  • HTTP

Time Collection

GET time from a single task

Obtains the array of times for a specific task by its id

  • Curl
  • HTTP
curl -X GET "{{url}}/times/{{id}}"
GET %7B%7Burl%7D%7D/times/%7B%7Bid%7D%7D HTTP/1.1
Host:

POST new timer to task by id

  • Curl
  • HTTP
curl -X POST -H "Content-Type: application/json" -d '{"timestamp": "Sat Aug 31 2019 08:36:35 GMT+0100", "duration": 666}' "{{url}}/times/{{id}}"
POST %7B%7Burl%7D%7D/times/%7B%7Bid%7D%7D HTTP/1.1
Host: 
Content-Type: application/json

{"timestamp": "Sat Aug 31 2019 08:36:35 GMT+0100", "duration": 666}
